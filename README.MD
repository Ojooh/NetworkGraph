# Network Graph
Network Graph is a network graph program that allows you to see and analyze the links between things, where an object might be a person, an event, a transaction, a car, or anything else. Entities are represented as nodes, and relationships between entities are shown as lines called links.

## Tools Used
[Pyhton 3.7.5](https://www.python.org/ftp/python/3.7.5/python-3.7.5.exe).

[VSCode](https://code.visualstudio.com/download)

## Installation
Network Graph is a desktop application built with python (tkinter) to help visualize the concep of graph and dikstra's algorithm using London station data set in a csv

* STEP 1: DOWNLOAD OR PULL NETWORK FOLDER
* STEP 2: OPEN TERMINAL AND CD INTO NETWORKGRAPH FOLDER
```bash
cd ...../~NetworkGraph_folder~/
```
* STEP 3: CREATE VIRTUAL ENVIRONMENT FOLDER
* first install virtualenv
```bash
pip install virtualenv
```
* Then run the below command
```bash
python -m venv env
```
* activate virtual environment
```bash
cd env/Scripts
```
```bash
activate
```
```bash
cd ../../
```
* STEP 4: INTSALL REQUIREMNTS.TXT FILE
```bash
pip install -r requirements.txt
```
* STEP 5: START PYHTON SERVER LOCALLY
```bash
cd src/
```
```bash
pyhton main.py
```
